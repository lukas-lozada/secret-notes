2.5 Absolute Extrema

Algorithm for Determining Extreme Values

Suppose that $f(x)$ is a continuous function over a closed interval [a, b].

To find the absolute maximum and minimum values of the function $f(x)$ on [a, b]:

  1. Find $f'(x)$.
  2. Determine the additional points to test in the interval [a, b].

    (that is, find all x values for which $f'(x)=0$ (critical points), $f'(x)$ does not exist (possible cusps, etc.).

  3. Create a table for (x, y) coordinates and list all $x$ values you plan to check: the endpoints $a,$ $b,$ and all critical values, possible cusps, etc.
  4. Find the function values of all the $x$ values in the table.
    1. The largest of these values is the absolute maximum of on the interval [a, b].
    2. The smallest of these is the absolute minimum of on the interval [a, b].
